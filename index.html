A lightweight, responsive framework.

Grid.

The grid is built on flex-box and has a max-width of 960px with a breakpoint at 480px. You can have any number of rows and columns and the grid will size them evenly. You can also nest rows within columns and the grid will not break. The max-width can be changed with one line of CSS and there is even a class to provide a full width overide. Using flex-box as the basis of the grid system allows for simple and clean coding.

Typography.

The font size is all set using rems and bases the root font-size at 62.5%. Using rems allows for better responsive support accross devices.

The typography base is Lato served by Google. It's set at 1.5rem over a 1.6 line height. All basic tags like anchors, strong, emphasis, and underline are included.

Headings are defined as well. Each with it's own letter-spacing, line-height, and margin/padding.

Buttons.

Buttons are included and come in 3 different flavours: plain, primary, and secondary. The colours can be configured in paper.css like all other coloured elements in Paper.
